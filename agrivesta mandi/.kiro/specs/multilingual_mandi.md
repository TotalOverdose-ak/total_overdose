# Multilingual Mandi Specification

## Goal
Empower rural vendors with AI to overcome language and price asymmetry.

## P0 Features (Implemented)
- Translation API integration.
- Voice input handling.
- Price data display.

## Future Roadmap
- WhatsApp integration for offline price alerts.
- Image recognition for vegetable quality grading.
- Regional dialect support (Bhojpuri, Marwari).
